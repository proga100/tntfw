Create and manage different member types easily for your BuddyPress site and allow different roles.

== Description ==

If you are planning to make a membership site with BuddyPress which includes different member types,
This is the plugin for you! <b>BP Member Type Manager</b> allows you to create and manage member types easily

= Features =

*   Create And Manage Member Types, e.g. Student/Teacher/Stuff  Doctor/Patient.
*   Allows Different Profile Fields for Different Member Types
*   Separate Registration Page for Different Member Types.
e.g.
example.com/register/student
example.com/register/teacher

*   Synchronize WordPress User Role with Member Types
*   More features are in development

Know detail features and development roadmap <a href="http://rimonhabib.com/buddypress-member-type-manager/"> from here </a>

You can develop a wide range of applications using BuddyPress with BuddyPress Member Type Manager plugin.
Some examples are, Social network based Learning Management System (involved Students, Teachers, Parents, Stuff).
Online Dating Sites (Male, Female).
Job Board sites (Employee, Employer)… Sky is the limit!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
